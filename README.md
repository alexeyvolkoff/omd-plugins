# omd-plugins

A collection of plugins for various note-taking platforms that integrate with [OnMyDisk](https://onmydisk.com) — a personal cloud/WebDAV storage solution.

## Available plugins

| Platform | Directory | Description |
|----------|-----------|-------------|
| Obsidian | [`obsidian/onmydisk-sync/`](obsidian/onmydisk-sync/) | Sync notes from Obsidian to OnMyDisk via WebDAV |

## Planned plugins

- UpNote
- Joplin
- Any other note-taking app with plugin support
